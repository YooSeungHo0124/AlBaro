## 🧾 Git Convention  

| 태그 | 의미 |
|------|------|
| `Feat` | 새로운 기능 추가 |
| `Fix` | 버그 수정 |
| `Refactor` | 리팩토링 |
| `Design` | UI 스타일 변경 |
| `Comment` | 주석 추가 또는 수정 |
| `Style` | 코드 포맷 변경 (기능 변화 없음) |
| `Test` | 테스트 코드 작업 |
| `Chore` | 설정, 빌드 관련 작업 |
| `Init` | 프로젝트 초기 설정 |
| `Rename` | 파일/폴더 이름 변경 |
| `Remove` | 파일 삭제 |
| `Docs` | 문서 작성/수정 |
